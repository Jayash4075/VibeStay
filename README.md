# 🌍 WanderLust

Welcome to **WanderLust**! This repository houses the development files, database structures, and REST API configurations for the WanderLust application.

---

## 📌 Repository Structure

To keep the production environment clean, this repository utilizes a multi-branch workflow:

* **`main`** (Default Branch): Contains repository documentation, configuration guides, and release notes. 
* **`database`**: Contains the active codebase, including database schemas, backend routing, and REST API development.

---

## 📂 Accessing the Code

The source code for this project is hosted on a separate development branch. To view the files or clone the active development environment, please switch branches:

### 🌐 On GitHub
Click the branch dropdown menu at the top left (currently showing `main`) and select **`database`** to browse the files.

### 💻 Via Command Line
If you are cloning this repository locally, you can fetch and switch to the development branch using the following commands:

```bash
# Clone the repository
git clone [https://github.com/YOUR_USERNAME/WanderLust.git](https://github.com/YOUR_USERNAME/WanderLust.git)

# Navigate into the project folder
cd WanderLust

# Switch to the active development branch
git checkout database
